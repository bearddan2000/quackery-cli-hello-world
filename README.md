# quackery-cli-hello-world

## Description
A POC for quackery programming language.

## Tech stack
- python

## Docker stack
- python:3.10-alpine3.18
- rzuckerm/quackery

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
